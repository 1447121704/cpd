Detailed Description

template<typename Transform, typename ProbabilityComputer>
class cpd::Runner< Transform, ProbabilityComputer >

Template class for running cpd registrations.

The two template arguments set the type of transform (e.g. Rigid) and the type of probability computer (e.g. FgtProbabilityComputer).
